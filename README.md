# pta-lp-measure
# Two-Dimensional Pulsar Distance Inference from Nanohertz Gravitational Waves by S.-R. Xiao, J.-Y. Song, Y. Shao, L.-F. Wang, J.-F. Zhang and X. Zhang

We introduce a two-dimensional method for inferring pulsar distances from the pulsar terms of nanohertz continuous gravitational-wave sources, and demonstrate its performance with simulated SKA-era pulsar timing array data.
